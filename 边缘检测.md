# 基于小波变换的边缘检测作业说明文档

## 1. 项目任务理解

本项目要求对 `samples` 中的样本图像进行边缘检测，并使用提供的 Ground Truth 计算定量评价指标。报告需要包含基本理论、算法步骤、中间结果、定性结果、定量结果、实验分析、总结和参考文献。

本项目的核心方法应围绕小波变换展开。Sobel、Canny 等传统方法可以作为对比实验，但不能替代主体算法。主体算法必须以小波分解得到的高频子带作为边缘响应的主要来源。

任务流程可以概括为：

```text
输入图像
-> 灰度化与预处理
-> 小波分解并提取高频子带
-> 构造边缘响应图
-> 多尺度融合与可选增强
-> 阈值化和后处理
-> 与 Ground Truth 对比
-> 计算 Precision、Recall、F1-score
```

当前项目中：

- 输入图像目录：`samples/images/`
- Ground Truth 目录：`samples/groundTruth/`
- 源代码目录：`src/`
- 输出结果目录：`results/`

## 2. 基本理论

### 2.1 边缘是什么

图像边缘通常对应灰度或颜色发生显著变化的位置。对于灰度图像 \(I(x,y)\)，可以用局部变化强度描述边缘响应：

\[
M(x,y)=\sqrt{I_x(x,y)^2+I_y(x,y)^2}
\]

其中，\(I_x\) 和 \(I_y\) 分别表示水平和垂直方向的变化率。\(M(x,y)\) 越大，说明该位置越可能是边缘。

### 2.2 小波为什么能检测边缘

二维离散小波变换可以将图像分解为低频和高频信息：

```text
低频：整体亮度、平滑背景、大尺度结构
高频：边缘、纹理、噪声、局部突变
```

一层二维离散小波变换可以表示为：

\[
I \xrightarrow{DWT} \{LL, LH, HL, HH\}
\]

其中：

- \(LL\)：低频近似子带，保留图像的整体结构；
- \(LH\)：水平或垂直方向的高频细节；
- \(HL\)：另一方向的高频细节；
- \(HH\)：对角方向的高频细节。

因此，小波边缘检测的核心思想是：利用高频子带中的局部突变信息构造边缘响应，再通过阈值化和后处理得到二值边缘图。

## 3. 整体算法流程

推荐流程如下：

```text
1. 读取输入图像 I 和 Ground Truth G
2. 将彩色图像转换为灰度图
3. 可选预处理：CLAHE、中值滤波、双边滤波、高斯滤波
4. 对灰度图进行一层或多层二维离散小波分解
5. 提取高频子带 cH、cV、cD
6. 构造每一层的小波边缘响应图
7. 将各层响应 resize 回原图大小
8. 进行多尺度融合
9. 可选轻量级梯度辅助增强
10. 可选非极大值抑制
11. 阈值化得到二值边缘图
12. 可选后处理：闭运算、开运算、去除小连通域、细化
13. 保存中间结果和最终边缘图
14. 计算 Precision、Recall、F1-score
```

这条流程比单独调用传统边缘检测函数更符合课程要求，也便于展示中间结果和分析参数影响。

## 4. 关键步骤说明

### 4.1 预处理

彩色图像通常先转换为灰度图：

\[
I_{gray}(x,y)=0.299R(x,y)+0.587G(x,y)+0.114B(x,y)
\]

预处理的目标是降低噪声、增强结构，同时尽量保留真实边缘。当前代码支持以下可选预处理方法：

- `--use-clahe`：使用 CLAHE 提升局部对比度；
- `--use-median-filter`：使用中值滤波抑制椒盐噪声；
- `--use-bilateral-filter`：使用双边滤波，在平滑噪声的同时尽量保留边缘；
- `--gaussian-kernel-size` 和 `--gaussian-sigma`：使用高斯滤波进行轻量平滑。

需要注意，预处理过强会抹掉弱边缘，因此应通过消融实验分析其影响。

### 4.2 小波分解

单层小波分解可以写作：

\[
(cA_1,cH_1,cV_1,cD_1)=DWT(I)
\]

多层分解是在上一层的低频近似子带 \(cA\) 上继续分解：

\[
(cA_j,cH_j,cV_j,cD_j)=DWT(cA_{j-1})
\]

一般来说，第 1 层更关注细节边缘，第 2 层更关注较粗的结构。层数过大会导致边缘定位变粗，因此本项目通常从 `level=1` 或 `level=2` 开始实验。

当前代码支持的小波基包括：

- `haar`
- `db2`
- `sym2`

### 4.3 边缘响应构造

对第 \(j\) 层高频子带，可以构造边缘响应：

\[
E_j(x,y)=\sqrt{cH_j(x,y)^2+cV_j(x,y)^2+cD_j(x,y)^2}
\]

其中：

- \(cH_j\)：水平细节子带；
- \(cV_j\)：垂直细节子带；
- \(cD_j\)：对角细节子带。

当前项目的主体边缘响应仍然来自小波高频子带。轻量级梯度辅助只用于增强响应，不作为核心检测方法。

### 4.4 多尺度融合

多层小波分解会得到不同尺度的边缘响应。各层响应需要先 resize 到原图大小，再进行融合：

\[
E(x,y)=\sum_{j=1}^{J}\alpha_j\tilde{E}_j(x,y)
\]

其中，\(\tilde{E}_j\) 是第 \(j\) 层响应图，\(\alpha_j\) 是融合权重。

当前代码支持：

- `--fusion-mode weighted`：加权融合；
- `--fusion-mode max`：最大响应融合；
- `--normalize-each-level`：融合前对每层响应进行归一化；
- `--level-weights`：手动设置多尺度权重，例如 `--level-weights 0.7,0.3`。

细尺度权重大时 Recall 可能提升，但误检也可能增加；粗尺度权重大时结果更稳定，但可能漏检弱边缘。

### 4.5 轻量级梯度辅助

为了提高边缘连续性，当前实现支持轻量级梯度辅助：

- `--use-gradient-assist`：启用梯度辅助；
- `--gradient-weight`：控制梯度辅助强度；
- `--gradient-mode multiply`：用梯度增强小波响应；
- `--gradient-mode blend`：将小波响应与梯度响应按权重融合。

需要强调：梯度辅助不是主体方法，最终响应仍以小波高频子带为核心。

### 4.6 阈值化与后处理

边缘响应图需要转为二值边缘图：

\[
B(x,y)=
\begin{cases}
1, & E(x,y)\geq \tau \\
0, & E(x,y)<\tau
\end{cases}
\]

当前代码支持三种阈值策略：

- `--threshold-method otsu`
- `--threshold-method fixed`
- `--threshold-method percentile`

也支持双阈值连接：

- `--use-hysteresis`
- `--no-hysteresis`
- `--low-threshold-ratio`

后处理支持：

- `--use-morphological-closing`：闭运算，连接断裂边缘；
- `--use-morphological-opening`：开运算，减少孤立噪点；
- `--min-object-size`：去除小连通域；
- `--use-thinning`：细化边缘。

后处理应适度使用。过强的后处理可能提升 Precision，但也可能删除真实弱边缘，降低 Recall。

## 5. 定量评价指标

设预测边缘图为 \(B\)，Ground Truth 为 \(G\)。逐像素比较时：

\[
TP=|\{B=1,G=1\}|
\]

\[
FP=|\{B=1,G=0\}|
\]

\[
FN=|\{B=0,G=1\}|
\]

对应指标为：

\[
Precision=\frac{TP}{TP+FP+\epsilon}
\]

\[
Recall=\frac{TP}{TP+FN+\epsilon}
\]

\[
F1=\frac{2\cdot Precision\cdot Recall}{Precision+Recall+\epsilon}
\]

可以这样理解：

```text
Precision 高：误检较少
Recall 高：漏检较少
F1-score 高：Precision 和 Recall 较均衡
```

边缘位置常有 1 到 2 个像素的偏移。如果使用容忍半径评价，需要在报告中明确说明；如果不使用，则说明采用严格逐像素比较。


## 7. 当前项目运行方式

安装依赖：

```bash
pip install -r requirements.txt
```

运行默认流程：

```bash
python -m src.run_all
```

推荐的一组增强配置：

```bash
python -m src.run_all --wavelet db2 --level 2 --threshold-method otsu --low-threshold-ratio 0.2 --use-gradient-assist --gradient-weight 0.15 --gradient-mode multiply --no-nms
```

输出目录：

```text
results/
  edges/          最终边缘图
  intermediate/   中间结果
  comparisons/    原图、预测边缘、Ground Truth 对比图
  metrics/        Precision、Recall、F1-score 指标表
```
